# Gesk
Este projeto foi uma solução de última hora para um problema na empresa em que eu trabalho (Auxiliar logístico na Böllhoff de Piracicaba). Visto que na filial que eu trabalho não havia um programa decente para impressão de etiquetas (8cm X 4cm) eu desenvolvi este app. Além de imprimir etiquetas ele tinha a capacidade de importar um arquivo .xml que vinha do programa Protheus 12 e fazia o cadastro automático de todos os itens e seus dados de nossa filial. Com isso o programa conseguia ajudar na localização do item em nosso estoque e fazia o vínculo com o código do cliente (pois trabalhamos com nosso código, mas fornecemos para um código diferente).

Este app não foi finalizado, comercializado ou distribuído! Apenas ultilizado pessoalmente por mim.
